## 잘된 점

- 검정·형광 녹색의 전술 UI가 세 화면에서 일관된다.
- 검색 버튼, 빠른 조회 탭, 승패 카드의 상태 구분이 분명하다.
- 검색창에는 접근 가능한 이름이 있고, 빠른 조회는 탭 역할과 선택 상태를 제공한다.
- 로딩 이후 실제 데이터가 안정적으로 표시되고 카드 확장 흐름이 짧다.

## 남은 위험과 다음 개선 후보

1. 상단 메뉴 링크가 `#`로 연결되어 실제 탐색 동작이 없고, 키보드 사용자에게도 목적지가 불분명하다.
2. 팀 기록은 시각적으로 표지만 의미 구조는 일반 요소 기반이다. 실제 `table` 또는 동등한 ARIA 행·열 구조가 필요하다.
3. 모바일 표의 9~12px 보조 글자는 정보 밀도에는 유리하지만 저시력 사용자에게 작을 수 있다. 긴 닉네임은 말줄임 처리되므로 전체 이름을 확인할 수단도 필요하다.
4. 키보드 포커스 순서, 포커스 표시, 스크린리더 발화, 색 대비 수치는 이번 스크린샷 중심 감사만으로 확정할 수 없다.
