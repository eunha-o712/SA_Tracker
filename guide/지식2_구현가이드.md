## DB 설계 (MySQL)

### clan_members 테이블
```sql
CREATE TABLE clan_members (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    user_name VARCHAR(50) NOT NULL COMMENT '서든어택 닉네임',
    clan_name VARCHAR(50) DEFAULT '모' COMMENT '클랜명',
    ouid VARCHAR(200) COMMENT 'NEXON API ouid (캐시용)',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    UNIQUE KEY uk_user_name (user_name)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 클랜원 INSERT 예시
```sql
INSERT INTO clan_members (user_name, clan_name) VALUES
('닉네임1', '모'),
('닉네임2', '모'),
('닉네임3', '모');
```

## 핵심 구현 코드 가이드

### NexonApiService
```java
@Service
public class NexonApiService {
    private final RestTemplate restTemplate;
    
    @Value("${nexon.api.key}")
    private String apiKey;
    
    private static final String BASE_URL = "https://open.api.nexon.com";
    
    private HttpHeaders createHeaders() {
        HttpHeaders headers = new HttpHeaders();
        headers.set("x-nxopen-api-key", apiKey);
        return headers;
    }
    
    // 닉네임 → ouid
    public String getOuid(String userName) {
        String url = BASE_URL + "/suddenattack/v1/id?user_name=" + 
                     URLEncoder.encode(userName, StandardCharsets.UTF_8);
        HttpEntity<String> entity = new HttpEntity<>(createHeaders());
        ResponseEntity<Map> response = restTemplate.exchange(url, HttpMethod.GET, entity, Map.class);
        return (String) response.getBody().get("ouid");
    }
    
    // ouid → 기본 정보
    public UserBasicDto getUserBasic(String ouid) { ... }
    // ouid → 계급 정보
    public UserRankDto getUserRank(String ouid) { ... }
    // ouid → 티어 정보
    public UserTierDto getUserTier(String ouid) { ... }
    // ouid → 최근 동향
    public UserRecentInfoDto getRecentInfo(String ouid) { ... }
    // ouid → 매치 목록
    public List<MatchDto> getMatches(String ouid, String matchMode, String matchType) { ... }
    // match_id → 매치 상세
    public MatchDetailDto getMatchDetail(String matchId) { ... }
    // 메타데이터 (계급/티어 이미지)
    public List<GradeDto> getGrades() { ... }
    public List<TierDto> getTiers() { ... }
}
```

### ClanService - ouid 캐싱 패턴
```java
@Service
public class ClanService {
    private final ClanMemberRepository repo;
    private final NexonApiService nexonApi;
    
    public String getOrFetchOuid(String userName) {
        ClanMember member = repo.findByUserName(userName).orElse(null);
        if (member != null && member.getOuid() != null) {
            return member.getOuid(); // DB 캐시 사용
        }
        String ouid = nexonApi.getOuid(userName);
        if (member != null) {
            member.setOuid(ouid);
            repo.save(member);
        }
        return ouid;
    }
}
```

### PlayerController
```java
@Controller
public class PlayerController {
    @GetMapping("/player")
    public String player(@RequestParam String name, Model model) {
        String ouid = clanService.getOrFetchOuid(name);
        model.addAttribute("basic", nexonApi.getUserBasic(ouid));
        model.addAttribute("rank", nexonApi.getUserRank(ouid));
        model.addAttribute("tier", nexonApi.getUserTier(ouid));
        model.addAttribute("recent", nexonApi.getRecentInfo(ouid));
        model.addAttribute("matches", nexonApi.getMatches(ouid, null, null));
        return "player";
    }
}
```

## 디자인 상세 가이드 (PPTX 기반)

### 참조 PPTX 디자인 특징
업로드된 PPTX 파일은 다음과 같은 시각적 특징을 가짐:

1. **배경:** 완전 블랙(#0D0D0D) - 밝은 배경 절대 사용 금지
2. **메인 색상:** 네온 그린/라임(#7ED957) - 제목, 숫자, 버튼, 링크에 사용
3. **보조 텍스트:** 흰색(#FFFFFF) 본문, 회색(#AAAAAA) 부가 설명
4. **장식 요소:** 좌하단에 작은 사각형 블록 패턴 (■초록 ■회색 ■흰색)
5. **배경 패턴:** 우측에 흩어진 기하학적 선(dash) 패턴 - 미세한 장식
6. **수치 강조:** 큰 통계 숫자는 네온 그린 + 굵은 이탤릭체로 크게 표시
7. **구분선:** 다크 그린(#3A5A2A) 얇은 라인으로 섹션 구분
8. **카드 프레임:** 네온 그린 테두리 (border: 1px solid #7ED957)

## 로딩 인트로 구현

업로드된 `Red_and_Black_Modern_Loading_YouTube_Intro.mp4` 파일을 사이트 첫 진입 시 인트로 영상으로 재생한다.

### 배치
- `src/main/resources/static/video/loading.mp4`에 MP4 파일 배치

### HTML 구현 (index.html 상단)
```html
<!-- 로딩 인트로 -->
<div id="loading-intro" style="position:fixed; top:0; left:0; width:100%; height:100%; 
     background:#000; z-index:9999; display:flex; align-items:center; justify-content:center;">
    <video id="intro-video" autoplay muted playsinline 
           style="max-width:100%; max-height:100%;">
        <source th:src="@{/video/loading.mp4}" type="video/mp4">
    </video>
</div>

<!-- 메인 콘텐츠 (처음에 숨김) -->
<div id="main-content" style="opacity:0; transition: opacity 0.5s;">
    ...메인 콘텐츠...
</div>
```

### JS 구현
```javascript
document.getElementById('intro-video').addEventListener('ended', function() {
    const intro = document.getElementById('loading-intro');
    const main = document.getElementById('main-content');
    intro.style.opacity = '0';
    intro.style.transition = 'opacity 0.5s';
    setTimeout(() => {
        intro.style.display = 'none';
        main.style.opacity = '1';
    }, 500);
});
```

- 영상 종료 후 fade-out → 메인 페이지 fade-in
- 세션 스토리지로 첫 방문만 인트로 재생, 이후 방문 시 스킵 가능
